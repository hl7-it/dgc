Profili FHIR

Profiles

This page lists all the profiles that have been specificed for the purpose of this project

Bundle (Digital Green Certificate)

This profile defines how to represent a vaccination certificate in FHIR by using a Bundle resource

Composition (Digital Green Certificate)

This profile defines how to represent a vaccination certificate in FHIR by using a Composition resource

Immunization (Digital Green Certificate)

This profile defines how to represent Immunizations in FHIR for building a Smart vaccination Card.

ImmunizationRecommendation (Digital Green Certificate)

This profile defines how to represent Immunization Recommandations in FHIR for building a Smart vaccination Card.

Location (Digital Green Certificate)

This profile defines how to represent Location in FHIR for building a Smart vaccination Card. This is used to describe optionally where the vaccination occured

Observation - Test Result (Digital Green Certificate)

This profile defines how to represent Test Results in a Digital Green Certificate.

Patient (Digital Green Certificate)

This profile defines how to represent Patient Data in a Digital Green Certificate.
